## 使用教學
1. 切換路徑 cd service-or-salary
2. 安裝套件 npm install
3. 執行環境 npm run dev

## 建立新分支
1. git branch -b 分支名
2. git checkout 分支名

## 編輯前，請先確認分支
1. git branch

## 新增頁面+使用連結
1. pages 資料夾，新增 jsx檔案
2. 撰寫 routers (App.jsx)
3. 加入連結 Link 

## 部屬網頁
1. npm run deploy:all
